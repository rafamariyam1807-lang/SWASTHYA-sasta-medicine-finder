# Swasthya – Affordable Medicine Alternative Finder

## Overview

Swasthya is a Flask-based web application developed for a healthcare hackathon. It helps users find affordable alternatives to branded medicines by comparing medicines with the same active ingredient and recommending the most cost-effective option.

The goal is to promote affordable healthcare and help patients make informed purchasing decisions.

---

## Problem Statement

Many patients unknowingly purchase expensive branded medicines because they are unaware of cheaper alternatives containing the same active ingredient. This increases healthcare expenses and limits access to affordable treatment.

---

## Solution

Swasthya allows users to:

-  Search for a medicine by brand name
- View its active ingredient
- Compare alternative brands with the same active ingredient
- Find the cheapest available alternative
-  Calculate monthly and yearly savings
-  Receive basic medicine-related health tips

---

## Features

- Smart Medicine Search
- Active Ingredient Detection
- Medicine Price Comparison
- Cheapest Alternative Recommendation
- Monthly & Yearly Savings Calculator
- Health Tips Based on Active Ingredient
- Fuzzy Search (supports minor spelling mistakes)
